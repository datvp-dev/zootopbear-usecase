Use Case Overview

Tài liệu liệt kê toàn bộ use case theo từng vai trò để khách hàng xác nhận đúng/đủ phạm vi trước khi đi vào đặc tả chi tiết. Seller thao tác trên hệ thống Zootop Bear; hệ thống thay phần file Excel riêng, gom đơn về danh sách đơn tổng nội bộ, tính giá riêng, xác nhận đơn rồi mới gửi sang xưởng phù hợp.
Seller
11 use case
Người dùng chính. Đăng ký, xem catalog, nạp tiền, tạo/import đơn theo template, theo dõi mã vận đơn, hủy đơn, tạo ticket.
Admin
11 use case
Full quyền: seller, catalog, giá, xưởng đang xử lý đơn mới, PrintBelle lưu lịch sử/import đơn cũ, đơn, ví, ticket, audit.
Kế toán
5 use case
Duyệt nạp tiền, điều chỉnh ví, xác nhận tiền đơn, xem chi phí xưởng, billing history, refund/đối soát.
Vận hành
6 use case
Kiểm tra file đơn tổng, xác nhận/gửi đơn sang xưởng, theo dõi trạng thái xưởng, mã vận đơn và đơn lỗi.
CSKH
4 use case
Xem seller/đơn/mã vận đơn được phân công, ghi chú hỗ trợ, tiếp nhận ticket và chuyển xử lý nội bộ.
