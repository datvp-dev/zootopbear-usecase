Tải ảnh mô phỏng/template thiết kế

Seller tải file mẫu sản phẩm, file hướng dẫn thiết kế hoặc template nhập đơn đúng với từng nhóm sản phẩm.

Wireframe dự kiến

Template theo sản phẩm

Luồng trực quan

Seller chọn sản phẩmChọn đúng nhóm hàng cần làm.
Xem file mẫuHệ thống hiển thị file thiết kế/import phù hợp.
Tải vềSeller tải file để chuẩn bị thiết kế hoặc nhập đơn.
Dùng khi tạo đơnFile thiết kế/ảnh mô phỏng được gắn vào đơn sau đó.

Ý hiểu hiện tại

  • Khách đã nói mỗi nhóm sản phẩm có thể cần template Excel riêng.
  • Template giúp seller nhập đúng dữ liệu để giảm lỗi khi import.
  • Chức năng này nên có, nhưng có thể không bắt buộc hoàn chỉnh ngay ngày đầu.

Cần hỏi khách

  • Phase 1 cần cho tải template thiết kế, template nhập đơn hay cả hai?
  • Template có cần version để biết seller đang dùng bản nào không?
  • Khi catalog thay đổi, template cũ có còn được import không?
