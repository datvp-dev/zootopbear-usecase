UC-SEL-06 · Seller

Import file đơn theo template

Seller upload file Excel theo template. Hệ thống kiểm tra dữ liệu từng dòng và gom đơn hợp lệ vào danh sách đơn tổng nội bộ.

Wireframe dự kiến

Import đơn · Tải template · Upload file

Luồng trực quan

Seller tải templateChọn đúng template theo nhóm sản phẩm.
Seller upload fileFile thay cho Excel riêng hiện tại.
Hệ thống kiểm traBắt lỗi thiếu địa chỉ, sai mã hàng, thiếu thiết kế.
Gom vào đơn tổngĐơn hợp lệ chờ nội bộ xác nhận.

Ý hiểu hiện tại

  • Hiện tại mỗi seller có một file Excel riêng.
  • Web mới nên thay bằng import file theo seller và theo template chuẩn.
  • Cần lưu batch import để đối chiếu nếu có lỗi.

Cần hỏi khách

  • Phase 1 bắt buộc seller dùng template chuẩn Zootop hay vẫn hỗ trợ file riêng từng seller?
  • Import lỗi một vài dòng thì cho nhập các dòng đúng trước hay bắt sửa cả file?
  • Seller có cần xem lịch sử từng lần import không?
