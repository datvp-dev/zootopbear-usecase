SCR-01
Đăng nhập / Đăng ký seller
Seller đăng ký tài khoản, chờ Admin duyệt, sau đó mới được dùng hệ thống.
Role: Seller, Admin
Zootop Bear

Use case gắn vào màn hình

UC-SEL-01UC-ADM-01
  • Seller điền form đăng ký và nhận trạng thái chờ duyệt.
  • Admin duyệt, từ chối, khóa hoặc gán tier cho seller.

Đơn mới · Import file · Ví · Ticket

  • Các nút chính dẫn tới tạo đơn, import đơn, nạp tiền và tạo ticket.
